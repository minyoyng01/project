# do it ; - Todo List Application

할 일 관리를 위한 모던 웹 애플리케이션입니다. Next.js와 TypeScript로 구축

## 🚀 주요 기능

### 할 일 목록 페이지
- **할 일 추가**: 검색바에 내용을 입력하고 `추가하기` 버튼을 클릭하여 새로운 할 일을 생성
- **할 일 목록 조회**: TO DO와 DONE 두 섹션으로 구분하여 표시
- **반응형 디자인**: 데스크톱, 태블릿, 모바일 화면에 최적화된 레이아웃

### 할 일 상세 페이지
할 일 항목을 클릭하면 상세 편집 화면이 인라인으로 표시됩니다:

- **항목 이름 수정**: 텍스트 입력 필드에서 할 일 제목 변경 가능
- **상태 변경**: 라디오 버튼으로 진행 중/완료 상태 토글
- **메모 추가**: 메모 텍스트 영역에 추가 정보 입력 가능
- **이미지 첨부**: 
  - 최대 1개의 이미지 파일 첨부 가능
  - 영문 파일명만 허용
  - 파일 크기 5MB 이하로 제한
  - 이미지 미리보기 제공
- **수정 완료**: 변경사항을 저장하고 목록으로 돌아가기
- **삭제하기**: 해당 할 일 항목 삭제

### 기타 기능
- **체크박스**: 할 일 완료/미완료 상태 토글
- **더블클릭 편집**: 할 일 제목을 더블클릭하여 빠른 편집
- **호버 액션**: 마우스 오버 시 편집/삭제 버튼 표시
- **로고 클릭**: 상단 로고를 클릭하면 페이지 새로고침

## 📦 기술 스택

- **Framework**: Next.js 16.1.5
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS v4
- **UI**: React 19.2.3

## 🎨 컴포넌트 설명

### `TodoInput`
검색바와 추가하기 버튼을 포함한 입력 컴포넌트입니다.
- 반응형 검색바 (280px/518px/1016px)
- 입력값이 있을 때만 추가 버튼 활성화
- Enter 키로 빠른 추가 가능

### `TodoSection`
TO DO 또는 DONE 섹션을 표시하는 컴포넌트입니다.
- 할 일 목록이 비어있을 때 안내 메시지 표시
- 반응형 이미지 표시 (Small/Large)

### `TodoItem`
개별 할 일 항목을 표시하고 편집하는 컴포넌트입니다.
- 체크박스로 완료 상태 토글
- 클릭 시 상세 편집 모드 표시
- 이미지 업로드 및 미리보기
- 메모 작성 기능
- 항목 수정 및 삭제

## 🎯 사용 방법

### 1. 할 일 추가하기
1. 상단 검색바에 할 일 내용 입력
2. `추가하기` 버튼 클릭 또는 Enter 키 입력
3. TO DO 섹션에 새 항목이 추가됨

### 2. 할 일 완료하기
- 할 일 항목의 체크박스를 클릭
- 완료된 항목은 DONE 섹션으로 이동

### 3. 할 일 상세 편집하기
1. 할 일 항목을 클릭
2. 상세 편집 화면에서:
   - 제목 수정
   - 상태 변경
   - 메모 추가
   - 이미지 첨부 (📷 아이콘 클릭)
3. `수정완료` 버튼으로 저장

### 4. 할 일 삭제하기
- 상세 편집 화면에서 `삭제하기` 버튼 클릭
- 또는 호버 시 나타나는 삭제 아이콘 클릭

## ⚠️ 주의사항

### 이미지 업로드 제한
- **파일명**: 영어(a-z, A-Z), 숫자(0-9), 특수문자(._-) 만 허용
- **파일 크기**: 최대 5MB
- **개수**: 항목당 1개만 첨부 가능

잘못된 파일을 선택하면 경고 메시지가 표시됩니다.
